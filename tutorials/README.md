# Tutorials

This section provides step-by-step guides and reproducible workflows for analyzing and visualizing *Drosophila* terminalia data.

## Current and Planned Tutorials
- **R/** — examples using Seurat, Monocle, or tidyverse tools  
- **Python/** — notebooks using Scanpy, pySCENIC, or other frameworks  
- **Visualization/** — tutorials for creating interactive plots and viewers (e.g., HCR/ISH image dashboards)

## Structure
Each tutorial folder may include:
- A Jupyter notebook (`.ipynb`) or RMarkdown file (`.Rmd`)  
- Sample data or links to datasets  
- A short `README.md` describing objectives and expected outputs

## Goals
- Provide well-documented, reproducible examples for educational use  
- Encourage adoption of best practices in single-cell and imaging analysis  
- Lower the barrier for users to explore FlyTerminalia data without heavy coding
